# Start

1. Install uv package manager [uv](https://docs.astral.sh/uv/getting-started/installation/)
    ```
    #linux
    curl -LsSf https://astral.sh/uv/install.sh | sh

    #windows
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```

1. Clone and run
    ```
    uv sync --all-extras
    ```
